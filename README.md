# Example implementation of an arquitecture to built a CRUD app.

## How to run

1. Install json server:
   ```bash
   npm i -g json-server
   ```
2. Run json server:
   ```bash
   json-server --watch db.json
   ```
3. Set up environment variables

   ```bash
   cp .env.example .env
   ```

4. Run development server
   ```bash
   yarn dev
   ```
